# Cryptanalyse de Vigenère

TME du cours Cryptographie — Licence Informatique, Sorbonne Université  
Travail en binôme, séances encadrées à la PPTI

---

## Description

Implémentation progressive du chiffrement de Vigenère et de sa cryptanalyse par analyse statistique.  
Le projet suit 9 étapes, chacune dans un fichier de test séparé, avec intégration continue GitLab automatisée à chaque push.

---

## Étapes du projet

| Fichier | Contenu |
|--------|---------|
| `test-1-cesar.py` | Chiffre de César (décalage simple) |
| `test-2-vigenere-cipher.py` | Chiffrement / déchiffrement Vigenère |
| `test-3-freq-IC.py` | Analyse de fréquences + Indice de Coïncidence |
| `test-4-decalages.py` | Estimation de la longueur de clé |
| `test-5-cryptanalyse-V1.py` | Première attaque par IC |
| `test-6-ICM-decalages.py` | Raffinement de l'estimation |
| `test-7-cryptanalyse-V2.py` | Reconstruction partielle de la clé |
| `test-8-correlations.py` | Corrélations statistiques |
| `test-9-cryptanalyse-V3.py` | Cryptanalyse complète automatique |

---

## Lancer les tests

```bash
# Tous les tests
./test-all.sh

# Un test spécifique
python test-3-freq-IC.py
```
