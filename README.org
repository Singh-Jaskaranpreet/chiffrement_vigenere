# Cryptanalyse Vigenère — Chiffrement & Attaque statistique (Python)

> Projet académique — Licence Informatique, Sorbonne Université (cours CRYPTO-2026)  
> Réalisé en **binôme** lors des séances de TME à la PPTI (créneau encadré)

---

## À propos du projet

Ce projet explore les **mécanismes de chiffrement classiques** et leur **cryptanalyse** à travers l'étude du chiffre de Vigenère. L'objectif pédagogique est double : comprendre les algorithmes de chiffrement symétrique historiques, et surtout maîtriser les méthodes statistiques permettant de les casser sans connaître la clé.

Le projet est découpé en **9 étapes progressives**, chacune dans un fichier de test séparé, avec intégration continue GitLab automatisée.

---

## Pipeline de cryptanalyse

```
Texte chiffré
    │
    ▼
test-1  → Chiffre de César (décalage simple, attaque par force brute)
    │
    ▼
test-2  → Chiffre de Vigenère (chiffrement / déchiffrement avec clé)
    │
    ▼
test-3  → Analyse de fréquences + Indice de Coïncidence (IC)
    │
    ▼
test-4  → Estimation de la longueur de clé par décalages
    │
    ▼
test-5  → Cryptanalyse V1 : attaque par IC sur sous-séquences
    │
    ▼
test-6  → ICM + décalages (raffinement de l'estimation)
    │
    ▼
test-7  → Cryptanalyse V2 : reconstruction de clé partielle
    │
    ▼
test-8  → Corrélations statistiques inter-colonnes
    │
    ▼
test-9  → Cryptanalyse V3 : reconstruction complète et déchiffrement automatique
```

## Concepts clés

### Chiffrement
- **César** : substitution monoalphabétique par décalage fixe
- **Vigenère** : substitution polyalphabétique avec clé cyclique

### Cryptanalyse (attaque sans clé connue)
- **Indice de Coïncidence (IC)** : mesure statistique de la distribution des lettres
- **Analyse de fréquences** : exploitation de la distribution typique du français
- **Estimation de la longueur de clé** : test de Kasiski / IC moyen
- **Corrélations** : comparaison entre sous-séquences pour reconstruire la clé

### Corpus de référence
Le fichier `germinal.txt` (roman de Zola) sert de référence statistique pour les fréquences de lettres en français — une approche réaliste de la cryptanalyse sur texte naturel.
